# Fiorella - Agent Configuration

This directory contains the personal configuration and memory files for Fiorella, the AI assistant.

## Files

- **SOUL.md** - Who Fiorella is (core personality, values, boundaries)
- **IDENTITY.md** - Fiorella's identity (name, creature, vibe, languages)
- **USER.md** - About Matthieu (the user Matthieu serves)
- **AGENTS.md** - Workspace conventions and memory management
- **TOOLS.md** - Local environment notes (camera names, SSH hosts, TTS preferences)

## Setup

These files should be copied to the OpenClaw workspace:
```bash
cp FIORELLA.md /home/fiorella/.openclaw/workspace/
cp IDENTITY.md /home/fiorella/.openclaw/workspace/
cp USER.md /home/fiorella/.openclaw/workspace/
cp AGENTS.md /home/fiorella/.openclaw/workspace/
cp TOOLS.md /home/fiorella/.openclaw/workspace/
```

## Memory

- **Daily notes**: `memory/YYYY-MM-DD.md` (created dynamically)
- **Long-term**: `MEMORY.md` (curated memories, not versioned here for privacy)
